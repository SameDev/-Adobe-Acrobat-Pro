Documentação da API

INTRODUÇÃO

Essa API é uma api de um ambiente real de um TCC (Trabalho de Conclusão de Curso) do Instituto federal catarinense. Para saber mais acesse esse link: github.

Você usuário que está acessando esse site não tem permissão para fazer Requisições a esta API!

Requisições HTTP permitidas:

POST: Criar e Atualizar recursos.

GET: Listar recursos.

DELETE: Deletar recursos.

Respostas HTTP:

  • 200 OK – a solicitação foi bem-sucedida (algumas chamadas de API podem retornar 201).
  • 201 Created – a solicitação foi bem-sucedida e um recurso foi criado.
  • 400 Bad Request - a solicitação não pôde ser compreendida ou faltavam parâmetros necessários.
  • 401 Unauthorized – falha na autenticação ou o usuário não tem permissões para a operação solicitada.
  • 404 Not Found – o recurso não foi encontrado.

REFERÊNCIAS

Usuário

São ouvintes e artistas que utilizaram do login e afins.

Atributos do Usuário:

  • id Int : indentificador único
  • email String : email único do úsuario
  • nome String : nome do úsuario
  • cargo String : cargo do úsuario (ADMIN, ARTISTA e USUARIO)
  • foto_perfil String : url da imagem de perfil do úsuario
  • data_nasc DateTime : data de nascimento do úsuario
